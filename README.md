# Phishing URL Detection

## Overview:
This project presents an intelligent phishing URL detection system using Machine Learning and Deep Learning techniques to classify phishing and legitimate websites. The project focuses on improving phishing detection accuracy through data preprocessing, model training, evaluation, and performance comparison.

## Objectives:
* Detect phishing URLs using AI techniques.
* Compare Machine Learning and Deep Learning models.
* Improve phishing detection accuracy.
* Enhance cybersecurity and user protection.

## Models:
* Logistic Regression
* Random Forest
* Artificial Neural Network (ANN)

## Data Preprocessing:
* Data cleaning
* Duplicate removal
* Dataset balancing using undersampling
* Feature scaling using Min-Max Scaling

## Performance Comparison

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 92.05%   |
| Random Forest       | 94.36%   |
| ANN                 | 93.22%   |

### Key Finding:
Random Forest achieved the best overall performance with 94.36% accuracy and 95.23% recall, making it the most reliable model for phishing URL detection in this project.
